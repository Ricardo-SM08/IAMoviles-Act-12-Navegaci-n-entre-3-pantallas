## Mi PROMPT

Lenguaje Dart Flutter, nivel principiante. Navegación entre 3 páginas utilizando rutas nombradas. Desde main llamar a la página 1, en AppBar de la página 1 mostrar el texto "Ricardo Montañez 6°J" en color Violeta, fondo claro e íconos negros. En body agregar un contenedor redondeado color orquidea(200 x 200) con texto negro y centrado además dde un botón de color negro con texto violeta  que redirija a la página 2. En la página 2 un appbar con texto ciruela("Segunda Página 6°J"), fondo claro, íconos negros. En body agregar una imagen desde la red y un botón para avanzar a la página 3. En la página 3 una AppBar con el texto "Página 3". Todo en un solo archivo.
